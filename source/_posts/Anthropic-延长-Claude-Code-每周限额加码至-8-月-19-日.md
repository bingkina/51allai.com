> Anthropic 将 Claude Code 每周使用限额提高 50% 的活动延长至 2026 年 8 月 19 日，覆盖 Pro、Max、Team 和传统按席位计费的 Enterprise 用户。活动自动生效，只适用于 Claude Code 的周限额，不改变 5 小时限额。
![Claude Code 每周限额提高 50% 并延长至 8 月 19 日](https://images.51allai.com/blog/claude-code-weekly-limit-august-cover_20260719_093454.png)

## 周限额提高 50% 延长到 8 月 19 日

此次活动从 2026 年 5 月 13 日持续到 8 月 19 日 23:59（太平洋时间）。适用用户的 Claude Code 每周使用限额在标准额度上提高 50%，不需要领取或修改账户设置。

8 月 19 日活动结束后，Claude Code 周限额恢复到标准水平，用户的套餐和账单不会因此改变。

## Pro、Max 和 Team 用户自动获得

活动覆盖 Pro、Max 和 Team 计划，也包括传统按席位计费的 Enterprise 用户。符合条件的账户会自动应用更高的周限额。

Free 计划和按量计费的 Enterprise 席位不在活动范围内。Enterprise 用户需要区分自己使用的是传统席位，还是按实际消耗计费的席位。

## 只调整 Claude Code 的周限额

提高 50% 的是 Claude Code 每周限额，适用于命令行工具、IDE 扩展、桌面端和网页版。Claude 网页端、桌面端、移动端以及 Claude Cowork 的使用限额不随此次活动改变。

Claude Code 的 5 小时限额也保持不变。周限额控制一个七天周期内的总用量，5 小时限额控制较短时间窗口内的用量；周限额增加后，单个 5 小时窗口不会随之扩大。

使用命令行版本的用户可以输入 `/usage`，查看账户当前的 5 小时限额、周限额和重置时间。
