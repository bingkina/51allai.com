> 阿里千问预告 Qwen3.8 即将发布并开放模型权重，参数规模为 2.4T。面向用户的 Qwen3.8-Max-Preview 已进入 Token Plan、Qoder 和 QoderWork，可先在阿里旗下订阅与智能编程、桌面办公产品中体验。
![Qwen3.8 2.4T 参数开放权重主题图](https://images.51allai.com/blog/qwen38-open-weights-cover-v1_20260719_170311.png)

## Qwen3.8 将以开放权重方式发布

Qwen3.8 的参数规模为 2.4T，也就是 2.4 万亿参数。参数是模型在训练中学到的数值，数量反映模型规模，但不能单独代表实际效果。

“开放权重”指训练完成后的模型参数文件对外提供，便于开发者在本地或自有服务器上运行和适配模型。它与同时公开训练代码、训练数据不是同一件事，因此本次预告更准确的说法是“开放模型权重”。

## Qwen3.8-Max-Preview 已开放体验

正式权重发布前，Qwen3.8-Max-Preview 已进入 Token Plan、Qoder 和 QoderWork。Token Plan 是阿里的模型订阅方案，用户可获取 API Key 和接口地址，配置到支持 OpenAI 或 Anthropic 协议的工具中使用。

Qoder 面向编程任务，提供桌面端、命令行和 JetBrains 插件；QoderWork 面向日常办公，可处理文件、数据和文档生成等任务。对想先测试模型的用户，这三个入口已可使用 Qwen3.8-Max-Preview。
